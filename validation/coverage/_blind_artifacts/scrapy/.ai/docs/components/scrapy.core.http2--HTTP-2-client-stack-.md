# Component: scrapy.core.http2 (HTTP/2 client stack)

Path: `scrapy/core/http2/` (agent.py, protocol.py, stream.py; `__init__.py` is empty).
Derived BLIND from source only.

## Role

A self-contained Twisted-based HTTP/2 client transport that pools per-`(scheme,host,port)` HTTP/2 connections, multiplexes each Scrapy `Request` onto an HTTP/2 stream, and fires a `Deferred[Response]` — built on the third-party hyper-h2 (`h2`) state machine, separate from the HTTP/1.1 path. The `H2Agent.request` entry point turns a `Request` into a pooled connection then a per-connection stream request — `scrapy/core/http2/agent.py:155-167`.

## Boundary rules

- Inbound-only consumer: this sub-package is imported solely by `scrapy.core.downloader.handlers.http2` (`from scrapy.core.http2.agent import H2Agent, H2ConnectionPool`) — `scrapy/core/downloader/handlers/http2.py:9`; the sub-package imports nothing from the downloader except the TLS context-factory helper (`scrapy/core/http2/agent.py:17`).
- ALPN enforcement: only `b"h2"` is acceptable; on a non-h2 negotiated protocol the connection is dropped without a GOAWAY — `scrapy/core/http2/protocol.py:139`, `scrapy/core/http2/protocol.py:271-283`, `scrapy/core/http2/protocol.py:482-483`.
- Hostname firewall: a stream refuses to send unless the request URL netloc matches the connection's base URI host/netloc/ip — else it closes with `INVALID_HOSTNAME` — `scrapy/core/http2/stream.py:199-207`, `scrapy/core/http2/stream.py:260-269`.
- Server-initiated stream ids are silently ignored (KeyError on unknown id) — `scrapy/core/http2/protocol.py:407-413`, `scrapy/core/http2/protocol.py:415-421`.

## Key facts

- Client stream ids are odd, generated by `itertools.count(start=1, step=2)` per RFC 7540 §5.1.1 — `scrapy/core/http2/protocol.py:117`.
- A connection is considered usable only after the remote ACKs SETTINGS: `h2_connected` AND `settings_acknowledged` gate `_send_pending_requests` — `scrapy/core/http2/protocol.py:155-162`, `scrapy/core/http2/protocol.py:181-189`, `scrapy/core/http2/protocol.py:423-428`.
- Concurrency cap = `min(local, remote) max_concurrent_streams`, recomputed as a property so a `RemoteSettingsChanged` is honored — `scrapy/core/http2/protocol.py:164-174`.
- Manual flow control: outbound data is chunked by `min(window, remaining)` then `min(chunk, max_outbound_frame_size)`; the rest is resent on `WindowUpdated` — `scrapy/core/http2/stream.py:285-323`, `scrapy/core/http2/protocol.py:450-456`.
- Idle timeout 240s via Twisted `TimeoutMixin`; on timeout the connection is closed with PROTOCOL_ERROR if streams are open else NO_ERROR, then errbacks `DownloadTimeoutError` — `scrapy/core/http2/protocol.py:87-88`, `scrapy/core/http2/protocol.py:324-348`.
- `StreamCloseReason` enum (8 values) is the single dispatch axis for how a stream resolves its response Deferred (ENDED→callback, RESET/CONNECTION_LOST/INACTIVE→`ResponseFailed`, MAXSIZE→`DownloadCancelledError`, CANCELLED→status 499 callback, INVALID_HOSTNAME→`InvalidHostname`) — `scrapy/core/http2/stream.py:58-83`, `scrapy/core/http2/stream.py:429-491`.
- Per-stream download limits prefer `request.meta` over spider/settings defaults; max-size is enforced both on the `Content-Length` header and on actual bytes received — `scrapy/core/http2/stream.py:115-120`, `scrapy/core/http2/stream.py:342-347`, `scrapy/core/http2/stream.py:373-376`.
- Hardening edge case: a literal `b"HTTP/2.0 405 Method Not Allowed"` reply (no proper h2 framing) is detected manually and raised as `MethodNotAllowed405`; `FrameTooLargeError` triggers a manual `abortConnection` because hyper-h2 does not drop the connection itself — `scrapy/core/http2/protocol.py:290-298`, `scrapy/core/http2/protocol.py:308-315`.
- hyper-h2 (`h2`) is a hard import dependency of this component but is NOT listed in `pyproject.toml [project].dependencies` (manifest dependencies begin at `pyproject.toml:9`, `cryptography` at `:11`, `twisted` at `:91`; no `h2`/`hyper` entry) — so HTTP/2 support is effectively an optional/transitive capability — `scrapy/core/http2/protocol.py:9-23`, `pyproject.toml:9-11`.
